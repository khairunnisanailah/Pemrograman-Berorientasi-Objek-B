## 1.	Naming convention   
Naming convention adalah suatu aturan yang mengatur penamaan variable, method, class, dan konstanta pada program java.  Tujuan utama naming convention adalah agar kode konsisten dan mudah dipahami.  
a.	Penamaan variable menggunakan camelCase dengan aturan:

-	Kata pertama menggunakan lowercase
-	Kata selanjutnya dimulai dengan huruf kapital
-	Tidak menggunakan karakter unik
-	Nama variable harus jelas dan konsisten

b. Penamaan boolean harus diawali dengan is, has, atau can

c. Penamaan konstanta menggunakan CONSTANT_CASE dengan aturan:
- Dideklarasikan dengan modifier final
- Seluruh huruf ditulis menggunakan kapital
- Setiap kata dipisahkan menggunakan (_)

d. Penamaan class menggunakan PascalCase dengan aturan:
- Tidak boleh menggunakan karakter unik
- Berupa kata benda
- Setiap kata dimulai dengan huruf kapital

e. Penamaan method menggunakan camelCase dengan aturan:
- Berupa kata kerja
- Nama dapat menjelaskan fungsi method
- Method berupa boolean memiliki aturan sama seperti penamaan boolean
## 2. Pentingnya naming convention  
Penamaan dalam pemrograman dinilai penting karena membuat kode lebih rapi dan terstruktur sehingga dapat membantu programmer lain ketika melakukan maintenance program. Penamaan juga dapat mempermudah kode untuk dibaca dan dipahami sehingga tidak adanya miskomunikasi ketika pengembangan program.
## 3.	Operasi Aritmatika  
Operator aritmatika digunakan untuk melakukan perhitungan matematika. Java akan melakukan operasi sesuai jenis dan urutan prioritas. Terdapat 3 jenis operasi aritmatika dalam java yaitu: compound, increment dan decrement, serta aritmatika dasar. Increment dan decrement yaitu x++, ++x, --x, dan x--, aritmatika dasar seperti +, /, *, -, dan %. Sementara compound yaitu +=, -=, *=, /=, %=. Terdapat pula aturan prioritas dengan urutan increment/decrement, perkalian/pembagian/modulus, penjumlahan/pengurangan, compound, dan terdapat tanda kurung untuk mengubah urutan.
## 4.	Contoh penerapannya  
Penerapan naming convention pada Latihan adalah pemberian aturan pada setiap variable, class, konstanta, dan Boolean sesuai dengan aturan masing-masing seperti contoh: isStatusTersedia ini dilakukan sesuai aturan Boolean, namaProduk dilakukan sesuai dengan aturan penamaan variable, dan PAJAK sesuai dengan aturan konstanta.  
Penerapan operasi aritmatika simbol + untuk penjumlahan, - untuk pengurangan, * untuk perkalian, / untuk pembagian. Khusus pembagian bisa menghasilkan nilai desimal maupun int dengan mengubah tipe data saat data ingin ditampilkan. Melakukan () untuk mengubah urutan dalam operasi matematika seperti dalam penghitungan pajak, agar menghitung jumlah pajak terlebih dahulu lalu diubah menjadi tipe data int.
## 5.	Kesulitan yang Dialami  
Kesulitan modul kurang lengkap sehingga masih harus mencari materi sendiri seperti menghitung pajak bagaimana mengubah tipe datanya.
